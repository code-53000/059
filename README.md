# Skate Map - 街头滑板图谱

一款专为街头滑手设计的本地记录应用，帮助你系统记录踩过的场地点位和动作练习进度，通过可视化城市图谱直观展示你的滑板地图。

## 功能特性

- 🗺️ **城市点位图谱**：Canvas 绘制的可缩放可拖拽地图，按类型标注场地
- 📍 **场地管理**：添加/编辑/删除场地，记录标签、路况、最佳时段
- 🎯 **动作记录**：记录每个动作的练习次数、成功与否、练习心得
- 🔗 **关联联动**：场地与动作通过练习记录关联，点击场地查看练过的动作
- 💾 **本地存储**：所有数据保存在浏览器本地，无需后端服务
- 🎨 **街头工业风**：深色主题 + 霓虹点缀，专属滑板文化美学

## 技术栈

- **前端框架**：Vue 3 (Composition API) + TypeScript
- **构建工具**：Vite 5
- **状态管理**：Pinia 2
- **样式方案**：SCSS + Tailwind CSS
- **绘图技术**：原生 Canvas 2D API
- **数据持久化**：localStorage
- **图标库**：Lucide Vue

## 架构分层

```

└─────────────┬───────────────────────────┘
              │
┌─────────────▼───────────────────────────┐
│  状态层 (State)                          │
│  Pinia Store - canvas / spot / trick     │
└─────────────┬───────────────────────────┘
              │
┌─────────────▼───────────────────────────┐
│  渲染层 (Render)                         │
│  CanvasRenderer - 网格 / 标记 / 交互     │
└─────────────┬───────────────────────────┘
              │
┌─────────────▼───────────────────────────┐
│  工具层 (Utils)                          │
│  坐标转换 / 本地存储 / ID 生成            │
└─────────────────────────────────────────┘
```

## 本地启动

### 环境要求

- Node.js >= 18.x
- npm 或 pnpm

### 安装依赖

```bash
npm install
```

```bash
npm run dev
```

```bash
npm run build
```

### 类型检查

```bash

```

### 代码检查

```bash
npm run lint
```

## 使用说明

### 浏览地图

- **拖拽平移**：按住鼠标左键拖动地图
- **滚轮缩放**：使用鼠标滚轮放大/缩小地图
- **缩放控件**：右下角按钮可精确缩放和重置视图

### 管理场地

1. 点击右下角「添加场地」按钮
2. 在地图上点击选择位置
3. 填写场地信息（名称、类型、标签等）
4. 点击侧边栏或地图上的场地标记查看详情

### 记录动作

1. 在侧边栏切换到「动作」标签
2. 点击「添加」创建新动作
3. 点击动作查看详情，可添加练习记录
4. 练习记录可以关联到具体场地

### 数据存储

所有数据自动保存到浏览器的 localStorage 中。清除浏览器数据会导致记录丢失，请定期备份重要数据。
